# Manipulación efectiva de Arrays en JavaScript

La rama `metodo_acceso` del repositorio se centra en la exploración y comprensión profunda de los métodos de acceso `entries()`, `keys()`, y `values()` de los arrays en JavaScript. Esta rama proporciona ejemplos prácticos y documentación detallada sobre cómo utilizar estos métodos para trabajar con arrays de manera eficiente.

## Pasos para comenzar

Para configurar el entorno y ejecutar el script desde la rama `metodo_acceso`, sigue estos pasos:

1. **Clona la rama metodo_acceso a tu máquina local**:

   ```bash
   git clone --branch metodo_acceso https://github.com/devsys-bit/arrays_js.git
   ```

2. **Navega al directorio del proyecto**:

   ```bash
   cd arrays_js
   ```

3. **Instala las dependencias necesarias**:

   ```bash
   npm install
   ```

4. **Inicia el script**:

   ```bash
   npm start
   ```

5. **Navega en el menú principal**:

   Una vez que el script esté en ejecución, se te presentará un menú con varias opciones. Para explorar los ejemplos de los métodos de acceso, selecciona la opción **7**, que corresponde a **Métodos de acceso**. Desde el submenú, podrás ejecutar y revisar ejemplos prácticos de `entries()`, `keys()`, y `values()`.

## Métodos cubiertos en esta rama

- **`Array.prototype.entries()`**: Devuelve un nuevo objeto de iterador de matriz que contiene pares clave/valor para cada índice de la matriz.
- **`Array.prototype.keys()`**: Devuelve un nuevo objeto de iterador de matriz que contiene las claves de índice para cada elemento en la matriz.
- **`Array.prototype.values()`**: Devuelve un nuevo objeto de iterador de matriz que contiene los valores para cada índice en la matriz.
